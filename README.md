# Portal Sign

### Usage:
Place down a sign with the following.

|         |
| :-------------: |
| [PORTAL]   |
| <worldname\> |
| <x\> <y\> <z\> |
| *<location name\> |
\* : Optional


Then place an iron pressure plate under the sign.

Boom you're done you can walk on the pressure plate to go to the location.
